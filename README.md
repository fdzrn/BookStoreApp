# BookStoreApp
Here some app's features :

#### Client Module
- Authentication
- Homepage
- Book list
- Book search
- Book detail
- Book checkout
- Add to favorite
- Payment via bank transfer (dummy)
- Purchase confirmation
- Purchase status

#### Admin Module
- Authentication
- Homepage
- Book management
- Book category management
- Client purchase history
